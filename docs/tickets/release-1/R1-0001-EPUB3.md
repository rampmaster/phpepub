# R1-EPUB3-01: Implementar packaging básico EPUB 3.0

Descripción

Implementar el empaquetado mínimo compatible con EPUB 3.0: `mimetype`, `META-INF/container.xml`, `OEBPS/book.opf` con atributos y prefijos necesarios, y soporte para `nav.xhtml` (HTML5 navigation document).

Criterios de aceptación
- El archivo `.epub` contiene `mimetype` con `application/epub+zip` en el primer entry sin compresión.
- `book.opf` tiene `version="3.0"` y contiene los metadatos esenciales (title, identifier, language).
- Existe un `nav.xhtml` y la entrada está presente en el `manifest` con `properties="nav"`.
- `epubcheck` (en CI) valida el paquete como EPUB 3.0.

Estimación: 3–4 días
